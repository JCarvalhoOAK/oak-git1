Quadras de tênis - Desafio
Introdução
Você foi encarregado de corrigir algumas implementações incorretas de uma API de back-end para uma plataforma de reserva simples para jogadores de tênis. Usando esta plataforma, os usuários podem fazer reservas em uma variedade de quadras de tênis cadastradas nela.

Este código já está desenvolvido no JUBYJUBY Clone-o, abra um branch de recurso.

O código real já tem as histórias de usuário abaixo implementadas. Você deve preencher pelo menos um dos mais ousados ​​para poder comparecer à entrevista. Caso você se sinta confortável para completar outros, isso contará na sua avaliação.

1. Como usuário, quero poder fazer uma reserva para uma ou mais quadras de tênis em uma determinada data programada

Como usuário, quero ser capaz de ver quais horários estão livres
Como usuário, quero poder cancelar uma reserva
Como usuário, quero poder reagendar uma reserva
5. Como administrador de quadra de tênis, quero ser capaz de criar / atualizar / excluir / localizar por id / localizar por nome / listar todos os convidados

Como administrador de quadra de tênis, desejo criar horários para uma determinada quadra de tênis
7. Como administrador de quadra de tênis, desejo cobrar um depósito de reserva de $ 10 do usuário, cobrado por quadra, que é devolvido após a conclusão da partida, para que os usuários não abusem de minha programação

Como administrador da quadra de tênis, desejo reembolsar o depósito da reserva se o usuário tiver cancelado ou reagendado a reserva com mais de 24 horas de antecedência
9. Como administrador de quadra de tênis, desejo manter 25% da taxa de reserva se o usuário cancelar ou reagendar entre 12:00 e 23:59 horas de antecedência, 50% entre 2:00 e 11:59 de antecedência, e 75% entre 0:01 e 2:00 de antecedência

Como administrador de quadra de tênis, quero manter 100% do depósito da reserva se o usuário não comparecer à reserva
Como administrador de quadra de tênis, quero ver um histórico de minhas reservas anteriores para poder usar as informações para melhorar a gestão do meu estabelecimento
